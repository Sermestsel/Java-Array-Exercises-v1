# Java Array Exercises v1.0

Java ile geliştirilmiş temel dizi analiz uygulaması.

## Proje Hakkında

Bu proje Java programlama dili kullanılarak geliştirilmiş bir konsol uygulamasıdır.

Diziler üzerinde temel algoritma ve veri analizi işlemleri uygulanmıştır.

## Özellikler

- Dizi elemanlarını analiz etme
- Toplam hesaplama
- Ortalama hesaplama
- Minimum değer bulma
- Maksimum değer bulma
- Çift ve tek sayı analizleri
- Temel döngü ve koşul yapıları kullanımı

## Kullanılan Teknolojiler

- Java
- IntelliJ IDEA
- Git
- GitHub

## Öğrenilen Konular

Bu proje kapsamında:

- Array (Dizi) kullanımı
- For döngüleri
- If-Else yapıları
- Metot kullanımı
- Boolean mantığı
- Temel algoritma geliştirme

konuları uygulanmıştır.
